# Contributing to Voyager AI

First off, thank you for considering contributing to Voyager AI! It's people like you that make this open-source community such a great place to learn, inspire, and create.

## Where do I go from here?

If you've noticed a bug or have a feature request, make sure to check our [Issues](../../issues) to see if someone else has already created it. 

## Fork & create a branch

If this is something you think you can fix, then fork Voyager AI and create a branch with a descriptive name.

## Implementing your fix or feature

At this point, you're ready to make your changes. Feel free to ask for help; everyone is a beginner at first.

## Running Tests

Before submitting a pull request, run all the tests locally:
```bash
make test
```

## Pull Request Process

1. Provide a clear description of the problem and the proposed solution.
2. Ensure you have added the necessary tests for your changes.
3. Make sure your PR passes all the CI checks.
